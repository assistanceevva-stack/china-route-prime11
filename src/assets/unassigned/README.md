# Непривязанные ассеты

Файлы в этой папке не импортируются в коде. Они перенесены сюда на Этапе 4 оптимизации из корня `assets/`.

- **Использование:** при необходимости привязать к блоку — перенести в `sections/<block>/` и обновить импорты в компонентах.
- **Удаление:** если файлы не нужны, их можно удалить из репозитория.
- В корне `assets/` остаются только: `shared/`, `sections/`, `unassigned/`, а также служебные папки `delivery-routes/`, `otzyvy/`, `videos/`, `fonts/` (шрифт дублируется в `shared/fonts/` для сборки).
